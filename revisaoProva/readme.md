# 📘 Fundamentos de Modelagem de Software

Este repositório contém um conjunto de materiais de estudo para a disciplina de **Modelagem de Software e Arquitetura de Sistemas - FECAP**.  

O objetivo é dar apoio para estudo e preparação para a avaliação, apresentando os principais conceitos trabalhados em aula de forma organizada, progressiva e textual.

Ao longo deste material, você encontrará explicações sobre o ciclo da engenharia de software, modelos de processo, abordagens ágeis e conceitos fundamentais relacionados a requisitos de software.

A recomendação é que a leitura seja feita na ordem apresentada abaixo, pois os conteúdos foram estruturados de forma sequencial, partindo de conceitos mais gerais até tópicos mais específicos.

---

## 🎯 Objetivo do Material

Este material foi elaborado com foco em revisão para avaliação, portanto:

- Prioriza clareza conceitual
- Apresenta exemplos contextualizados
- Destaca diferenças importantes entre conceitos
- Reforça pontos frequentemente cobrados em provas

---

## 🧠 Orientação de Estudo

Para um melhor aproveitamento:

- Leia os textos com atenção, buscando entender os conceitos (e não apenas memorizar)
- Tente relacionar os conteúdos entre si
- Reflita sobre exemplos práticos apresentados em aula
- Revise os conceitos mais importantes antes da prova

---

## 🚀 Considerações Finais

A modelagem de software é uma etapa essencial no desenvolvimento de sistemas, pois permite compreender problemas, estruturar soluções e garantir que o software atenda às necessidades dos usuários.

Dominar esses conceitos é fundamental não apenas para a prova, mas também para a formação como profissional de tecnologia.

Bons estudos!
